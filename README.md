# PC-8001 ミサイルコマンダー用パッチ

## 目的
* アーケード版に近づけるための修正や機能の追加

## 更新履歴
* 2026/03/22 githubで公開

## 主な内容
* オール機械語化
* PCG-8100による効果音
* タイトル画面と終了画面を修正
* ハイスコア登録機能
* スマートボムを追加

## 必要なもの
* ゲーム本体（アイ・オー1981年2月号掲載 P.153）

## 手順
* ゲームの機械語部分のみをモニタからロードする（BASIC部分は不要です）
* パッチプログラム `patch.cmt` をモニタからロードする
* `G9000`でゲームが開始する
